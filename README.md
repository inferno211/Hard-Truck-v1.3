# Hard-Truck-v1.3

Przedstawiam wam mój stary gamemode który nosi nazwę HardTruck map v1.3. Jest to kontynuacja poprzedniej wersji v1.0 (temat z HardTruck map v1.0). 

Co posiada gamemode: 
-System automatów 
-System fotoradarów 
-Koguty na pojazdy 
-Prywatne pojazdy 
-System domów 
-System załadunków 
-System 3d textów 
-System AFK 
-System prawa jazdy 
-System zarządzania kontem
